Kredy - Gizlilik Politikası / Privacy Policy

Son güncelleme / Last updated: 2026-03-18

1. Veri Toplama / Data Collection

TR: Kredy - Bütçe & Finans uygulaması, aşağıdaki verileri toplar:

EN: Kredy - Budget & Finance app collects the following data:

  • Kişisel Bilgiler / Personal Information: Ad (Name), E-posta adresi (Email address) — hesap oluşturma ve kimlik doğrulama için / for account creation and authentication
  • Finansal Bilgiler / Financial Information: İşlem geçmişi (Transaction history) — harcama takibi ve bütçe yönetimi için / for expense tracking and budget management
  • Fotoğraflar / Photos: Fiş/fatura fotoğrafları (Receipt photos) — AI fiş tarama özelliği için, kısa süreli işlenir / for AI receipt scanning, processed ephemerally
  • Uygulama Bilgileri / App Information: Kilitlenme günlükleri (Crash logs) — uygulama performansını izlemek için / for monitoring app performance
  • Cihaz Kimlikleri / Device Identifiers: Push bildirim token'ları (Push notification tokens) — bildirim göndermek için / for sending notifications

2. Veri Kullanımı / Data Usage

TR: Toplanan veriler yalnızca uygulama işlevselliği, hesap yönetimi ve analiz amaçlı kullanılır. Verileriniz üçüncü taraflarla paylaşılmaz.

EN: Collected data is used solely for app functionality, account management, and analytics. Your data is not shared with third parties.

3. Veri Güvenliği / Data Security

TR: Tüm veriler aktarım sırasında şifrelenir (TLS/SSL). Firebase Authentication güvenli kimlik doğrulama sağlar.

EN: All data is encrypted during transit (TLS/SSL). Firebase Authentication provides secure identity verification.

4. Veri Saklama / Data Storage

TR: Finansal verileriniz öncelikli olarak cihazınızda (localStorage) saklanır. İsteğe bağlı olarak Firebase Realtime Database ile bulut yedekleme yapılabilir.

EN: Your financial data is primarily stored on your device (localStorage). Optional cloud backup is available via Firebase Realtime Database.

5. Hesap Silme / Account Deletion

TR: Hesabınızı ve ilişkili tüm verilerinizi uygulama içinden Menü > Hesap Ayarları > Hesabı Sil bölümünden silebilirsiniz. Silme işlemi geri alınamaz.

EN: You can delete your account and all associated data from Menu > Account Settings > Delete Account within the app. Deletion is irreversible.

6. Veri Silme Talebi / Data Deletion Request

TR: Belirli verilerinizin silinmesini talep etmek için g.alpatar@gmail.com adresine e-posta gönderin.

EN: To request deletion of specific data, email g.alpatar@gmail.com.

7. Üçüncü Taraf Hizmetler / Third-Party Services

  • Firebase Authentication — kimlik doğrulama / authentication
  • Firebase Realtime Database — veri senkronizasyonu / data sync
  • Firebase Cloud Messaging (FCM) — push bildirimleri / push notifications
  • Google Gemini API — fiş tarama (fotoğraflar kısa süreli işlenir) / receipt scanning (photos processed ephemerally)
